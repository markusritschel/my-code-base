My Code Base

This is a collection of routines that I’ve developed over time and that I use in my daily work.

Installation

Clone this repo via

git clone https://github.com/markusritschel/my-code-base

Then, in the new directory (cd my-code-base/) install the package via:

python setup.py install

or via

python setup.py develop

if you plan on making changes on the code.

Testing

To test the code, run make test in the source directory. This will execute both the unit tests and docstring examples (using pytest).

Run make coverage to generate a test coverage report and make lint to check code style consistency.
