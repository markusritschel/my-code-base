# Installation

## Installation via pip
The easiest way to install the package is via pip directly from this repository:

```bash
$ pip install git+https://github.com/markusritschel/my_code_base.git
```

## Clone repo and install locally
Alternatively, you can clone the repository from github and install the package locally:

```bash
$ git clone git+https://github.com/markusritschel/my_code_base.git
$ cd my_code_base
$ pip install . -e
```
